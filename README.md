# React Chess

- Implement Chess game without Chess library
- To study React hooks
- Reduce HOC pattern
- Reduce ternary operator
- Test driven development
- The goal is implements, not for completing

## Installation & Start

```bash
npm install
npm start
```

open browser, http://localhost:3000
